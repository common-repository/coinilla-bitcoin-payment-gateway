Accept bitcoin on your website and get payments instantly in your own wallet.

== Description ==

Note: This plugin works with WooCommerce plugin ONLY!

Coinilla is a bitcoin payment gateway. We provide all tools you need to accept bitcoin on your website. We have ready to use plugins for you to start accepting bitcoin in minutes!

You have full control of your money and privacy. We do not provide online wallet for your payments. All payments go to your own wallet almost instantly.

== Installation ==

1. Install WooCommerce plugin for WordPress.
2. In WordPress admin select `Add New` in `Plugins` menu.
3. Select `Upload Plugin` and upload `coinilla-plugin.zip` file.
4. Activate Plugin.
5. Select `Settings` from `WooCommerce` and click `Checkout` tab.
6. select `Coinilla` and setup it.
